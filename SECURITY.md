# Security Policy

## Supported Versions
Only the latest version has security updates.

## Reporting a Vulnerability
If you find a vulnerability, please report it in the "Issues" tab. For this purpose, you can use our `Vulnerability report` template, and don't forget to add the `vulnerability` label to your post (it is automatically assigned if you use the template). Or, if you are a developer, you can fork this repository and create a pull request.\
\
If we confirm that the vulnerability is present, we will make sure to fix it ASAP and ship it in the next update.
